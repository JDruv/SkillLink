# 🔗 SkillLink - In sviluppo

Benvenuto nel repository della landing page ufficiale di **SkillLink**, un progetto personale, piattaforma PWA dedicata allo scambio delle competenze. 
L'idea alla base del progetto è semplice ma rivoluzionaria: *la conoscenza è la nostra valuta*. "Io insegno a te, tu insegni a me".

## 📖 Il Progetto

Questo repository contiene il codice sorgente della landing page di presentazione del progetto. È stata sviluppata con un design moderno, animazioni fluide e una UI in stile "Glassmorphism" per catturare l'attenzione dei futuri utenti e invitarli a unirsi alla lista d'attesa.

### ✨ Funzionalità Core (App in Sviluppo)
* 🤝 **Baratto Puro:** Nessun pagamento in denaro. Scambia le tue abilità (es. lezioni di chitarra in cambio di programmazione Python).
* 🏆 **Gamification:** Sistema di reputazione con Badge "Sensei", trofei rari e "Coins" virtuali per premiare l'impegno.
* 🛡️ **Profili Verificati:** Autenticazione sicura tramite Google e sistema di feedback per garantire affidabilità.

## 🧠 Il Motore di Matching

L'applicazione (attualmente in sviluppo) utilizzerà un potente algoritmo su database per collegare gli utenti:
1.  🥇 **Perfect Match:** Corrispondenza bidirezionale ("Io offro ciò che cerchi E tu offri ciò che cerco").
2.  🥈 **One-Way:** Corrispondenza unidirezionale, ideale per spendere i "Coins" accumulati insegnando.
3.  🥉 **Discovery:** Esplorazione libera di talenti nella propria zona o online.

## 🛠️ Tecnologie Utilizzate

### Landing Page (Questo repository)
* **Angular e Scss:** Layout semantico, CSS Grid/Flexbox, variabili CSS e Glassmorphism.
* **Js/Ts:** Animazioni allo scroll (IntersectionObserver), logica della modale e menu mobile responsivo.
* **Canvas API:** Sfondo animato interattivo con particelle e connessioni generate in real-time.

### Tech Stack Previsto per la Web App
* **Frontend:** Angular 17+ & Ionic UI (PWA)
* **Backend & DB:** Supabase (PostgreSQL)
* **Autenticazione:** Google Auth

## 🚀 Come visualizzare la Landing Page

Trattandosi di un sito web statico, non hai bisogno di installare dipendenze o server complessi.

1. **Clona o scarica** questo repository sul tuo computer.
2. Apri la cartella del progetto.
3. avvia con npm run dev
4. La landing page si aprirà nel tuo browser predefinito!
